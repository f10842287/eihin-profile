【私について】

私は鄧(トウ)榮(エイ)彬(ヒン)と申します。現在、台北科技大学の情報工学科の修士課程を卒業しました。私は運動が好きで、特にランニングが大好きです。学生時代にはよく外でランニングをして体を鍛え、気分をリフレッシュしていました。普段は日本のドラマやアニメ、美食のビデオを見るのが好きです。

【学歴】

>>高等専門学校時代<<

高等専門学校時代、私はウェブデザインの選手を務め、訓練の過程で HTML5 (HTML、CSS、Javascript)、PHP、jQuery、Laravel の基本的な使い方を学びました。この時からウェブデザインに興味を持ち始め、全国技能競技大会のウェブデザイン職種で佳作を受賞しました。順位はそれほど良くありませんでしたが、自分にはまだ成長の余地があることを知りました。

普段は日本のドラマやアニメを見るのが好きで、日本文化に興味を持ち、高等専門学校時代に学校が開設した日本語検定クラスに参加し、初めてJLPT試験を受け、JLPT N5の資格を取得しました。大学時代は独学し、1年生と2年生でそれぞれ JLPT N2、JLPT N1 の資格を取得しました。

>>大学時代<<

他のプログラミング言語の基礎知識も学びました。主に Java、Python、ASP.NET を学びました。授業で学んだため、非常に深くはありませんが、将来関連する言語を使用する際には再度研究を行います。当時、3年生の時に行ったプロジェクトは、PHP 言語を使用し、Laravel フレームワークを用いて作成した内部ウェブサイトシステムで、大学の農業検査センターと協力したプロジェクトでした。このウェブサイトは主に農業検査センターの紹介と農産物検査報告の生成を含んでいます。

>>大学院(修士)<<

修士課程では、1年目にソフトウェア工学関連のコースを先に修習しました。当時、ちょうど研究室が資策会と協力したプロジェクトがあり、このプロジェクトを早めに引き受け、臨時エンジニアとして約3ヶ月間務めました。主に Django を使用してウェブサイトを作成しました。修士2年目には他のプロジェクトを担当し、論文の執筆を始めて卒業しました、担当したプロジェックは主に JAVA と PHP 言語でシステムのメンテナンスしています。この過程で学んだことは、問題を解決する際に一連のプロセスを整理する方法と、未経験の問題に直面した際に自己学習を通じて関連知識を探し出し、問題を解決する方法です。これは現在も自分が努力している部分です。

【将来の計画】

私はウェブデザイナーやソフトウェアエンジニアに興味があるため、将来のキャリア発展においても関連知識に触れることを希望しています。フロントエンドとバックエンドの両方に少し触れていますが、現在はこの部分に関連する職を探しています。学んだことの大部分は学校の授業や独学であり、将来、実際のプロジェクト開発において素早く習得し、慣れることを望んでいます。

自家附近文化介紹

中和烘爐地

中和烘爐地是在中和區這邊蠻有名的一座土地公廟，主要除了來這裡拜拜的人，因為地理位置可以看到整個台北盆地，又因為靠近山上的關係，常有觀光客和登山客來這裡爬山和拜拜。

一水牛排

一水牛排是這一兩年剛在新北市中和區這邊建立的牛排館，目前只有一個總店，採用自助式Buffet，有水果、麵包、玉米濃湯、咖啡、紅茶及碳酸飲料等餐點，最低消費價大約150，算是CP值挺高的，蠻推薦吃這家。

興南夜市

這個夜市位於新北市中和區，相較於其他夜市比較沒有那麼大，算是小型夜市，因為附近有國中和高職的關係，所以每天晚上人流通量都蠻多的。
